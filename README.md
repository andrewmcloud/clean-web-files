# clean-windows-files

Removes '%20' from windows files and replaces with '-'. Usesful when working with Windows generated files on linux or Mac. 

## Installation

    $ lein uberjar

## Usage

    $ java -jar clean-windows-files-0.1.0-standalone.jar -d /path/to/directory

## Options

`-d, --directory`

`-h, --help`
